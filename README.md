# X UI Tailwind

A responsive X (Twitter) UI clone built using HTML and Tailwind CSS.

## Features

- Responsive design
- Mobile bottom navigation
- Tablet icon-only sidebar
- Desktop full sidebar
- Sticky sidebar layout
- Tweet feed UI
- X-inspired design

## Tech Stack

- HTML
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/Rishabh-09-eng/x-ui-tailwind
```

Install dependencies:

```bash
npm install
```

Run Tailwind:

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

Open:

```text
index.html
```

in browser.
